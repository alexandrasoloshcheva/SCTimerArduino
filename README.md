# SCTimerArduino
A Rubiks Cube Timer for Arduino

## Hardware
* Arduino UNO
* Display MELT MT-16S2H-2FLA
* Red led
* Green led
* 3 buttons
* 5 220Ohm resistors
* Breadboard
* Jumpers Male to Male
